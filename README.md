# Minesweeper

## Description

#### _**This is a clone of the classic Minesweeper game, using Angular.**_

#### _**By Tyler Stephenson, June 16, 2017**_

### Setup/Installation Requirements
* clone <link to repo>
* cd <local repo>

* run this series of commands:
  * `$ npm install`
  * `$ ng serve`
